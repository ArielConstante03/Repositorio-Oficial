### 2. **Conversor de Unidades** (`conversor_unidades.py`)
Programa completo para convertir entre diferentes tipos de unidades.

#### 🎯 Características:
- ✅ Conversión de **PESOS**: gramos, kilogramos, libras, onzas, toneladas
- ✅ Conversión de **LÍQUIDOS**: mililitros, litros, galones, onzas líquidas, tazas, pintas
- ✅ Conversión de **GASES**: volumen en metros cúbicos, litros, centímetros cúbicos
- ✅ Conversión **CRUZADA**: líquido ↔ peso (usando densidades de sustancias)
- ✅ **Historial** de conversiones realizadas
- ✅ Menú interactivo intuitivo

#### Uso:
```
python conversor_unidades.py
```

#### Ejemplos de Conversiones:

**Pesos:**
- 5 kg → gramos: 5000 g
- 10 libras → onzas: 160 oz

**Líquidos:**
- 1000 ml → litros: 1 l
- 5 galones → litros: 18.927 l

**Gases (Volumen):**
- 1 m³ → litros: 1000 l
- 500 cm³ → ml: 500 ml

**Conversiones Cruzadas:**
- 1 litro de agua → gramos: 1000 g (densidad agua = 1.0 g/ml)
- 100 gramos de aceite → mililitros: 108.7 ml (densidad aceite = 0.92 g/ml)

#### Sustancias Disponibles (para conversiones cruzadas):
- 💧 Agua (densidad: 1.0)
- 🫒 Aceite (densidad: 0.92)
- 🥛 Leche (densidad: 1.03)
- 🍷 Alcohol (densidad: 0.789)
- 🌡️ Mercurio (densidad: 13.6)
- ⛽ Gasolina (densidad: 0.72)
- 💨 Aire (densidad: 0.0012)

#### Unidades Disponibles:
